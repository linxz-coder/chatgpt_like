# 安装
1.**安装所需依赖项**  
terminal 输入：
``` 
pip install -r requirements
```
<br>

2.**本地部署**  
terminal输入：
```
python3 server.py
```  
如果是旧版本pyhon，将"python3"替换成"python"
<br><br>

3.**进入网页**  
在浏览器输入:
```
http://127.0.0.1:5000
```
网页出现了！
